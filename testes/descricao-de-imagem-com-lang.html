Teste de descrição de imagem com múltiplos idiomas

Esta página apresenta diferentes formas de fornecer descrições de imagem acessíveis a leitores de tela, contemplando conteúdo em dois idiomas (português e inglês). O foco é demonstrar como usar o atributo aria-describedby para associar uma descrição rica e multilíngue à imagem, mantendo a acessibilidade.

Exemplo 1 – aria-describedby com <figure> e <figcaption> (oculta)

Neste exemplo, a imagem é envolvida pela estrutura semântica HTML5 <figure> e <figcaption>. A legenda é visualmente escondida usando a classe sr-only, mas permanece acessível aos leitores de tela via aria-describedby.

Exemplo 2 – aria-describedby com <div> em vez de <figure>

Aqui usamos uma <div> comum em vez de <figure>. A descrição continua associada via aria-describedby, e o conteúdo multilíngue permanece acessível por meio da marcação com lang. Essa abordagem evita que o leitor de tela anuncie elementos estruturais como "figura" e "legenda", tornando a leitura mais direta.

Capa do livro com o título Test of Image description with multiple languages and accessibility features for screen readers.

Exemplo 3 – alt parcial + aria-describedby com detalhamento multilíngue

Este exemplo combina uma descrição breve no atributo alt (que evita que a imagem seja anunciada como "gráfico sem etiqueta" por alguns leitores de tela como o NVDA) com um detalhamento adicional multilíngue via aria-describedby. É a solução mais recomendada, pois une clareza imediata e descrição completa.
