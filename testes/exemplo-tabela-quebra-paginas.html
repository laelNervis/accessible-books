Tabela Longa com quebra de páginas

Como cada página contém a numeração no início, a tabela deve ser dividida em duas partes. Para que o usuário do leitor de tela saiba que é uma continuação da tabela, o caption deve ser repetido na segunda parte da tabela, como um texto oculto. Os textos ocultos são lidos pelo leitor de tela, mas não são exibidos na tela.

Neste exemplo, os textos foram ocultos usando a classe CSS "sr-only", definida da seguinte forma: 

            .sr-only {
                position: absolute;
                width: 1px;
                height: 1px;
                padding: 0;
                margin: -1px;
                overflow: hidden;
                clip-path: polygon(0 0, 0 0, 0 0);
                white-space: nowrap;
                border: 0;
            }
