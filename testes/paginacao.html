Teste para numeração da página

O objetivo deste teste é avaliar formatos para exibição do número da página, incluindo a leitura da palavra "página" antes da numeração apenas para os leitores de tela.

Exemplo 1

X
Página X

Neste exemplo, X é inserido em um span com o atributo role="textbox" e o atributo aria-labelledby="pagina-label" referencia o id do elemento que contém o texto "Página X". 

                    <span role="textbox" aria-labelledby="pagina-label">X</span>
                    <div id="pagina-label" class="sr-only" aria-hidden="true">Página X</div>

Exemplo 2

X

Neste exemplo, X é inserido em um span com o atributo role="textbox" e o atributo aria-label="Página X". 

                    <span role="textbox" aria-label="Página X">X</span>

Exemplo 3

Página X

Neste exemplo, apenas a palavra "Página" é inserida em um span oculto por CSS. O atributo role="text" é usado para indicar que o elemento contém texto. Isso evita que o leitor de tela interrompa a leitura sequencial. 

                    <span role="text" class="sr-only">Página</span> X

                    .sr-only {
                        position: absolute;
                        width: 1px;
                        height: 1px;
                        padding: 0;
                        margin: -1px;
                        overflow: hidden;
                        clip-path: polygon(0 0, 0 0, 0 0);
                        white-space: nowrap;
                        border: 0;
                    }
